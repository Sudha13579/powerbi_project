🥤 Coca-Cola Sales Analysis Dashboard (Power BI)
📌 Project Overview

This project analyzes Coca-Cola sales data using Power BI to gain insights into sales performance, profit trends, and regional distribution.

The dataset was cleaned and transformed before building interactive dashboards to help understand business performance and support data-driven decision making.

The dashboard provides insights into:

Sales performance by region

Profit analysis

Beverage brand performance

Monthly and yearly trends

Distribution channel analysis

🎯 Objectives

The main objectives of this project are:

Analyze sales performance of Coca-Cola products

Identify top performing regions and products

Track profit and revenue trends

Create interactive dashboards for business insights

Practice data cleaning, transformation, and visualization

🛠️ Tools & Technologies Used

Power BI – Data visualization and dashboard creation

SQL – Data cleaning and transformation

Excel – Initial data exploration

GitHub – Project documentation and version control

📂 Dataset Information

The dataset contains sales information for Coca-Cola products, including:

Retailer

Beverage Brand

Region

Sales Amount

Profit

Units Sold

Date

Distribution Channel

The dataset was cleaned using SQL queries to remove inconsistencies and prepare it for visualization.

🔄 Data Cleaning Process

Data cleaning steps performed using SQL:

Removed duplicate records

Handled missing values

Standardized column names

Converted date formats

Corrected inconsistent data entries

Filtered unnecessary columns

The Power BI dashboard includes the following visualizations:

📈 Total Sales Analysis

📊 Units Sold by Product
